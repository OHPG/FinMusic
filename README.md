# FinMusic
`FinMusic`是一款在`HarmonyOS NEXT`上的`Jellyfin`音乐播放器。

## 主要功能
* 支持多服务器、多地址、多用户。
* 支持媒体中心、通知栏，快捷操作。
* 支持夜间模式。
* 自适应布局，适配手机、平板等设备。

## 技术架构
* 开发语言：ArkTS
* UI框架：ArkUI

## 设备要求
* HarmonyOS NEXT 5.0.3(15) 及以上。
